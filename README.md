# Ricky & Morty App
App developed with Kotlin and Jetpack Compose. Uses https://rickandmortyapi.com/ to show Rick & Morty data.

-------------------------------------------

###### Architecture
- MVVM

###### Design
- Jetpack Compose

###### Network
- Retrofit

###### Dependency injection
- Hilt

###### Images
- Coil

###### Other dependencies
- Compose Navigation
- Hilt Navigation
